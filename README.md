# jazzurlifeup
website

"Jazzurlifeup" - Jazz Spence Portfolio
A curated collection of creative work spanning writing, producing, directing, editing, and illustration. This website is a custom-built "organized chaos" designed to showcase a decade of creative skill-honing.

Features
Custom Sidebar Navigation: A fixed navigation bar for seamless browsing.

Responsive Project Pages: Dedicated project mini-pages with integrated YouTube video embeds for film work.

Art Gallery: An interactive artwork showcase featuring custom hover overlays and lightbox viewing.

Dynamic Design: Features a unique decorative sidebar strip and hand-drawn UI elements.

Built With
HTML5 & CSS3: Semantic structure and custom Flexbox layouts.

JavaScript: Lightweight scripts for lightbox functionality.

SFTP: Deployed and managed via VS Code SFTP integration.

Project Structure
/CSS: Contains the master stylesheet.

/artwork: Storage for project thumbnails and gallery pieces.

/artwork/assets: UI design elements like logos, favicons, and decorative strips.

/photogallery: High-resolution featured imagery.
